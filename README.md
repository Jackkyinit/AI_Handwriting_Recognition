# AI Handwriting Recognition (AI-DEC Project)

This project recognizes handwritten text using the Microsoft TrOCR Transformer model.

## Features

- Upload handwritten image
- AI extracts text
- Built with Streamlit

## Run Project

- pip install -r requirements.txt
- streamlit run streamlit_app.py
